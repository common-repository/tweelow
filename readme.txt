== Description ==
What is "Tweelow"? Tweelow is a WordPress plugin that helps you to make few (for now ;)) things with twitter API.

**For more info:**

* Visit official page of <a href="http://creative-boy.com/">Tweelow Plugin.</a>

== Features ==

* Allows you to display the count of your twitter followers.
* Allows you to display your latest tweet.
* Adds a ReTweet button to your latest status

== Installation ==
Installation is quite easy.

**Installation**

1. Download  the plugin from the plugin's official page or from WordPress. Upload them to your wordpress plugin folder (wp-content/plugins/).
2. Activate the plugin from WordPress administration panel.

**Activation**

3. Go to Tweelow Settings in you WordPress administration panel.
4. Write your username. No password required!
5. Click Save!

**Finishing**

6. Now you will need to work a bit with your Theme. Open wherever you want to display your followers' count or your last tweet.

For count of your followers, paste this code wherever you want (between PHP tags):

display_tweelow();

For latest tweet, paste this code:

lateststatus_tweelow();

You're done! Now have fun with Tweelow!
